## Installation
```bash
npm install -g @marsidev/create
```

## Usage
`create <files> [options]`

## Examples
```bash
create index.ts
```

```bash
create src/index.ts
```

```bash
create index.html style.css app.js
```

```bash
create src/index.html src/styles/style.css src/app.js
```

```bash
create --base src/components Navbar.tsx Footer.tsx
```
